Аннотация

В работе представлена интеллектуальная система для автоматической оценки кредитного риска на основе методов машинного обучения. Разработана модель классификации на основе алгоритма Gradient Boosting, обученная на исторических данных кредитных заявок. Система использует следующие входные параметры: возраст заёмщика, годовой доход, статус жилья, опыт работы, параметры кредита (сумма, процентная ставка, доля дохода на кредит), кредитную историю и кредитную оценку. Модель способна предсказывать вероятность дефолта по кредиту с высокой точностью. Достигнуты следующие метрики качества: точность (Accuracy) 93.8%, ROC-AUC 0.949, F1-Score 0.842, Precision 95.2%, Recall 75.4%. Разработано веб-приложение на платформе Streamlit для практического использования системы, обеспечивающее интерактивную оценку кредитного риска с визуализацией результатов и бизнес-интерпретацией. Система может быть использована в образовательных целях для изучения применения машинного обучения в финансовой аналитике и банковском деле.

Ключевые слова: машинное обучение, классификация, Gradient Boosting, кредитный риск, оценка дефолта, финансовые технологии, веб-приложение, Streamlit

Введение

Современные технологии машинного обучения находят широкое применение в различных областях, включая финансовую аналитику и банковское дело [1, 2]. Одной из важных задач в этой области является автоматическая оценка кредитного риска и предсказание вероятности дефолта заёмщика [3, 4]. Традиционные методы оценки кредитоспособности основаны на экспертных оценках и статистических моделях, которые требуют значительного времени и могут быть субъективными. Применение методов машинного обучения позволяет автоматизировать процесс оценки кредитного риска, повысить точность и скорость принятия решений [5, 6].

Актуальность данной работы обусловлена необходимостью разработки эффективных инструментов для автоматизации процесса кредитного скоринга в условиях роста объемов кредитных заявок и требований к скорости принятия решений в финансовых организациях [7].

Целью данной работы является разработка интеллектуальной системы для оценки кредитного риска на основе алгоритмов машинного обучения и создание веб-приложения для практического использования системы в образовательных и исследовательских целях.

Материалы и методы

Описание данных

Для обучения модели использовались исторические данные кредитных заявок из открытого датасета Credit Risk Dataset [8]. Исходные данные включали следующие признаки:

Личные данные заёмщика:
- person_age – возраст заёмщика (лет);
- person_income – годовой доход (доллары США);
- person_home_ownership – статус жилья (MORTGAGE, OWN, RENT, OTHER);
- person_emp_length – опыт работы на текущем месте (лет).

Параметры кредита:
- loan_amnt – сумма кредита (доллары США);
- loan_int_rate – процентная ставка (%);
- loan_percent_income – доля дохода, приходящаяся на выплату по кредиту (от 0 до 1);
- loan_intent – цель кредита (DEBTCONSOLIDATION, EDUCATION, HOMEIMPROVEMENT, MEDICAL, PERSONAL, VENTURE);
- loan_grade – кредитная оценка (A, B, C, D, E, Other).

Кредитная история:
- cb_person_default_on_file – наличие дефолтов в прошлом (0/1);
- cb_person_cred_hist_length – длина кредитной истории (лет).

Целевая переменная loan_status представляла собой бинарную классификацию: 0 – кредит возвращен (Non-default), 1 – дефолт (Default).

Предобработка данных

На этапе предобработки данных были выполнены следующие операции:

1. Анализ пропущенных значений: выявлены и обработаны пропуски в данных.
2. Обработка выбросов: применены методы обнаружения и обработки аномальных значений.
3. Преобразование признаков:
   - Логарифмирование признака person_income для нормализации распределения;
   - Применение Box-Cox преобразований для нормализации числовых признаков (person_age, person_emp_length, loan_amnt, loan_int_rate, loan_percent_income, cb_person_cred_hist_length);
   - Масштабирование числовых признаков с использованием RobustScaler для устойчивости к выбросам [9];
   - Кодирование категориальных признаков: OneHotEncoder для person_home_ownership и loan_intent, OrdinalEncoder для loan_grade [9].

4. Разделение данных: данные были разделены на обучающую и тестовую выборки с сохранением пропорций классов.

Методы машинного обучения

Для решения задачи бинарной классификации кредитного риска были протестированы следующие алгоритмы:

1. Логистическая регрессия (Logistic Regression) – базовый линейный классификатор [6];
2. K-ближайших соседей (K-Nearest Neighbors, KNN) – метод на основе сходства;
3. Дерево решений (Decision Tree) – интерпретируемый алгоритм;
4. Случайный лес (Random Forest) – ансамблевый метод на основе деревьев решений [6, 9];
5. Градиентный бустинг (Gradient Boosting) – ансамблевый метод с последовательным улучшением [7, 10].

Все модели были объединены в pipeline с использованием библиотеки scikit-learn [9] для обеспечения воспроизводимости и удобства развертывания.

Оптимизация гиперпараметров

Для выбора оптимальной модели и гиперпараметров использовался метод оптимизации с библиотекой Optuna [11]. В качестве метрики оптимизации использовался ROC-AUC, учитывающий способность модели различать классы.

Лучшие результаты показал алгоритм Gradient Boosting Classifier со следующими гиперпараметрами:
- n_estimators: 240
- max_depth: 8
- learning_rate: 0.164
- subsample: 0.954
- min_samples_split: 2
- min_samples_leaf: 3

Результаты

Производительность модели

Обученная модель Gradient Boosting достигла следующих показателей на тестовой выборке:

- Accuracy (Точность): 93.8%
- ROC-AUC: 0.949
- F1-Score: 0.842
- Precision (Точность предсказаний): 95.2%
- Recall (Полнота): 75.4%
- Specificity (Специфичность): 98.9%

Высокие показатели метрик свидетельствуют о хорошей способности модели различать кредитоспособных и некредитоспособных заёмщиков. Особенно важным является высокий показатель Precision (95.2%), что означает, что когда модель рекомендует отказать в кредите, она в большинстве случаев права. Это критически важно для минимизации финансовых потерь банка.

Матрица ошибок показывает следующие результаты:
- Правильные одобрения (True Negatives): 3759
- Ложные отказы (False Positives): 40
- Пропущенные дефолты (False Negatives): 261
- Найденные дефолты (True Positives): 802

Разработка веб-приложения

Для практического использования разработанной модели было создано веб-приложение на платформе Streamlit [12], предоставляющее следующие возможности:

1. Интерактивная оценка риска: пользователь может ввести данные заёмщика через удобную форму с валидацией входных значений и получить оценку кредитного риска с вероятностями дефолта и возврата кредита.

2. Визуализация результатов: интерактивные графики, показывающие распределение вероятностей исходов, выполненные с использованием библиотеки Plotly.

3. Бизнес-интерпретация: автоматическая генерация рекомендаций (одобрить/отказать) с обоснованием на основе вероятности дефолта и анализом ключевых факторов риска.

4. Информационная панель: отображение метрик качества модели, статистики предсказаний и технических характеристик системы.

5. Образовательные материалы: разделы с объяснением метрик модели, принципов работы системы и интерпретацией результатов для пользователей без технического образования.

Приложение имеет интуитивно понятный интерфейс, не требует специальных знаний в программировании и может быть использовано как специалистами финансовых организаций, так и студентами для обучения применению машинного обучения в финансовой аналитике.

Обсуждение

Разработанная система демонстрирует эффективность применения методов машинного обучения для решения задачи оценки кредитного риска [1, 3, 4]. Использование алгоритма Gradient Boosting позволило достичь высокой точности предсказания благодаря способности модели обрабатывать нелинейные зависимости между признаками и устойчивости к выбросам в данных [7, 9, 10].

Важным аспектом работы является правильная предобработка данных, включающая логарифмирование дохода, применение Box-Cox преобразований для нормализации распределений и использование RobustScaler для масштабирования признаков [2, 5]. Эти преобразования улучшили качество модели и её способность обобщать знания на новые данные.

Включение различных типов признаков (числовых, категориальных, порядковых) в модель позволило учесть комплексное влияние факторов на кредитоспособность заёмщика. Особенно важными оказались такие признаки, как кредитная история, доля дохода на кредит и кредитная оценка, что согласуется с практикой кредитного скоринга [3, 4].

Разработанное веб-приложение делает систему доступной для практического использования и может служить образовательным инструментом для изучения применения машинного обучения в финансовой аналитике [1, 6]. Интерактивный интерфейс позволяет пользователям экспериментировать с различными сценариями и понимать влияние различных факторов на оценку кредитного риска.

Следует отметить, что модель показывает высокую специфичность (98.9%), что означает низкий уровень ложных отказов. Однако показатель Recall (75.4%) указывает на то, что модель пропускает около 24.6% дефолтов. Это может быть улучшено настройкой порога принятия решения или использованием дополнительных признаков.

Заключение

В работе успешно разработана интеллектуальная система для оценки кредитного риска на основе исторических данных кредитных заявок. Модель Gradient Boosting показала высокую точность классификации (Accuracy 93.8%, ROC-AUC 0.949) и способность эффективно различать кредитоспособных и некредитоспособных заёмщиков.

Созданное веб-приложение на платформе Streamlit обеспечивает удобный интерфейс для использования системы в практических и образовательных целях. Система может быть применена в финансовых организациях для автоматизации процесса кредитного скоринга и в образовательных учреждениях для демонстрации возможностей машинного обучения в финансовой аналитике.

Перспективы дальнейшего развития системы включают:
- Расширение набора признаков для улучшения точности (например, данные о занятости, семейном положении);
- Использование более сложных алгоритмов (например, нейронных сетей или ансамблей моделей);
- Добавление функций объяснимости модели (SHAP values, feature importance);
- Интеграция с базами данных кредитных историй;
- Разработка системы непрерывного обучения на новых данных.

Annotation

The paper presents an intelligent system for automatic credit risk assessment based on machine learning methods. A classification model based on the Gradient Boosting algorithm was developed, trained on historical credit application data. The system uses the following input parameters: borrower age, annual income, home ownership status, employment length, loan parameters (amount, interest rate, debt-to-income ratio), credit history and credit grade. The model can predict the probability of loan default with high accuracy. The following quality metrics were achieved: accuracy 93.8%, ROC-AUC 0.949, F1-Score 0.842, Precision 95.2%, Recall 75.4%. A web application on the Streamlit platform was developed for practical use of the system, providing interactive credit risk assessment with visualization of results and business interpretation. The system can be used for educational purposes to study the application of machine learning in financial analytics and banking.

Keywords: machine learning, classification, Gradient Boosting, credit risk, default prediction, financial technology, web application, Streamlit
